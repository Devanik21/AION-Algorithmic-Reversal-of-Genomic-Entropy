# AION API Reference

## src.models
* `GeneticRegulatoryNetwork`: Core class for GRN logic.
* `Phenotype`: Represents the physical state of the organism.

## src.simulation
* `SimulationEngine`: Manages ticks and population dynamics.
* `EnvironmentGrid`: Spatial resource management.

## src.analysis
* `calculate_shannon_entropy`: Quantifies GRN disorder.
